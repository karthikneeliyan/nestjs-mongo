## Description

API's Exposed

Create Company

Create Team (Should have company ID in path)

GET Company by ID

Search company by the name

Get All Teams (Return all teams as an array grouped within company object)


## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
## Run with docker

```bash
# development
$ docker compose up dev


# production mode
$ docker compose up prod
```
